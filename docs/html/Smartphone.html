Smartphone module

class Smartphone.Smartphone(brand: str = '', model: str = '', operating_system: str = '', monti_version: str = '', id: str = '')[Quellcode]

Bases: object

Ein Teilnehmer einer Studie kann sich mit einem Smartphone registrieren. Daten wie: _Marke_, _Softwareversion_, _ID_, … werden in dieser Klasse gespeichert. Das wichtigste ist die ID, da diese zur Authentifizierung des Teilnehmers verwendet wird. Es dürfen somit nicht zwei Smartphones mit der gleichen ID registriert werden können.
